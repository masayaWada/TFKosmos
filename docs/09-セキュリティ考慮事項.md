# 9. セキュリティ考慮事項

### 9.1 認証情報の扱い

- 認証情報はメモリ上のみ、永続化しない
- 環境変数や設定ファイルに保存しない
- ログに認証情報を出力しない

### 9.2 機密情報の除外

- AccessKey Secret、パスワード、MFAシードは取得しない
- Terraformコードに機密情報を含めない

### 9.3 ローカル実行

- すべての処理はローカルで実行
- 外部サービスへのデータ送信なし
- 生成ファイルはローカルファイルシステムに保存

### 9.4 入力検証

- すべてのユーザー入力を検証
- パストラバーサル攻撃を防ぐ
- SQLインジェクション対策（該当する場合）
